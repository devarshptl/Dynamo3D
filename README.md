# Dynamo3D

## Implementation

### Insert Mode (i)
1) 1 - add cube
2) 2 - add bumpy cube
3) 3 - add bunny

### Translation Mode(o)
1) w - Postitive x-axis
2) a - Negative x-axis
3) s - Postitive y-axis
4) d - Negative y-axis
5) q - Postitive z-axis
6) e - Negative z-axis

### Rotation Mode (o)
1) f - Postitive x-axis
2) g - Negative x-axis
3) h - Postitive y-axis
4) j - Negative y-axis
5) k - Postitive z-axis
6) l - Negative z-axis

### Scale Object (o)
1) , - Zoom plus
2) . - Zoom minus

### Refraction and Dynamic Environment Mapping (o)
1) b - activate

### Other Mode
1) y - light mode
3) s - shadow
4) a and d - move the light

### Environment Mapping
1) v - Mirror shading
2) c - phong shading

### Camera Control(u)
1) w - Postitive x-axis
2) a - Negative x-axis
3) s - Postitive y-axis
4) d - Negative y-axis
5) q - Postitive z-axis
6) e - Negative z-axis
